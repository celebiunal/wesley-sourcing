# Wesley Sourcing Website

Official website for **[Wesley Sourcing](https://wesleysourcing.com/)** — Your Trusted China Sourcing Partner in Yiwu.

## About

Based in Yiwu, China's largest wholesale market, Wesley Sourcing connects European buyers with Chinese manufacturers. With 13+ years of experience, we handle product sourcing, quality control, supplier management, and logistics.

**Company:** Shanghai Wesley Import & Export Co., Ltd. (上海维思乐进出口有限公司)

## Tech Stack

- Static HTML5, CSS3, JavaScript (vanilla)
- Hosted on Cloudflare Pages
- Contact form via Formspree
- Google Fonts (Inter)

## Deployment

Deployed via Cloudflare Pages. The site is a static HTML site — no build step required. Simply deploy the root directory.
